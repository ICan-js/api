MobileNet V1 (Howard, 2017)

MobileNet é um modelo de rede neural convolucional leve, por conta da utilização da técnica de convolução profunda separável. No ICan.js este modelo foi retreinado com um conjunto de dados de Libras captados com a ajuda de 9 colaboradores. O artigo da MobileNet está disponível em aqui.

Abaixo é exemplificado como consumir este modelo utilizando Tensorflow.js

Primeiro carregue o Tensorflow.js em seu código HTML


<script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@0.15.1/dist/tf.min.js"></script>

Feito isto, basta adicionar em seu script o código abaixo para o carregamento do modelo


const model = tf.loadModel("https://ican-api.herokuapp.com/models/mobilenetv1/model.json");
